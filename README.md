# nightdream
an Android Daydream clock

# Preparations
 - Download the sdk

https://developer.android.com/studio/index.html

 - Build Tools 23.0.3
 - API Level 23 (Android 6.0)

# Build

```shell
$ git clone https://github.com/firebirdberlin/nightdream.git
$ cd nightdream
$ git submodule init
$ git submodule update
$ make debug
```
